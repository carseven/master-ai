2.3. DBSCAN¶

DBSCAN (Density-Based Spatial Clustering of Applications with Noise) es un algoritmo de clustering creado en 1996, no paramtrico, basado en densidades, y que marca como outliers aquellos puntos que no superan un umbral de densidad establecido. Constituye uno de los algoritmos de clustering ms usados y citados, y tambin es relativamente eficiente cuando se usan estructuras de datos adecuadas para su implementacin (pudiendo llegar a ser de tiempo O(n2) en el peor caso).

In [ ]:
fitMe = X.to_numpy()

# Compute DBSCAN, eps (epsilon) determina la mxima distancia entre dos muestras para considerarlas vecinas
# En este escenario queremos forzar un resultado binario para poder determinar que casos son especiales de los no especiales, por lo tanto la epsilon es un valor grande
# En caso de querer detectar otros subconjuntos se puede conseguir con una epsilon ms pequea
db = DBSCAN(eps=20000, min_samples=1)
y_hc = db.fit_predict(X)
labels = db.labels_

# Number of clusters in labels, ignoring noise if present.
n_clusters_ = len(set(labels)) - (1 if -1 in labels else 0)
n_noise_ = list(labels).count(-1)

print('DBSCAN labels', labels)
